# Forced Displacement Tracker : South Sudan

Interactive Streamlit dashboard for exploring displacement patterns, contributing conditions, and regional variation across South Sudan.

## Live Dashboard

Open the deployed dashboard here:

https://info288-south-sudan-dashboard.streamlit.app/

## Project Files

- `app.py` — main Streamlit application
- `data/` — dashboard datasets
- `assets/` — images and visual assets
- `requirements.txt` — Python dependencies

## Run Locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the Streamlit app:

```bash
streamlit run app.py
```

## Dashboard Focus

The dashboard is organized around:

- displacement patterns over time
- contributing conditions such as conflict, food insecurity, prices, and flooding
- overlapping pressures summarized through CSI and regime views
